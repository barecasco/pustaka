Lecture Notes




Databricks is built to make Big Data simple. Apache Spark made a big step towards achieving this mission by providing a unified framework for building data pipeline. Databricks takes this further by providing a zero-management cloud platform built around spark that delivers:

  1. Fully managed spark clusters
  2. An interactive workspace for exploration and visualization.
